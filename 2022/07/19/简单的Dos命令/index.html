简单的Dos命令

打开cmd的方式

  1. 打开+系统+命令提示符
  2. win+r 输入cmd打开控制台
  3. 在任意文件夹下，按住shift+鼠标右键点击，在此处打开命令窗口
  4. 资源管理器地址栏前加上cmd

常用的dos命令

#盘符切换 
dir #查看当前目录下所有文件 
cd change directory #切换目录 
cd .. #返回上一级
cls #清理屏幕（clear screen）
exit #退出终端
ipconfig #查看ip信息
md #创建文件夹
cd> #新建文件
del #删除文件
rd #移除目录
